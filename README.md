# r-universe
repo to enable ALA packages to appear on https://r-universe.dev/. r-universe improves discoverability of R packages.

See https://atlasoflivingaustralia.r-universe.dev/builds for our institutional page on r-universe. Only packages listed in `packages.json` will appear.

You can use badges from r-universe on your package documentation; see https://atlasoflivingaustralia.r-universe.dev/badges. Examples:
